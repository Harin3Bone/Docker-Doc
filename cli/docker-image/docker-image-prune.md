# docker image prune

### Synopsis

```
docker image prune
```

### Options

* \-a
* \--all
  * Remove all unused images
* \--filter
  * Provide filter values
* \--force
* \-f
  * Do not prompt confirmation

### Commands

* Remove unused image

```
docker image prune
```

* Remove all unused image

```
docker image prune -a
```

* Remove unused image without confirmation

```
docker image prune -f
```
